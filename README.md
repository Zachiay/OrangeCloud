# OrangeCloud
This is a pigtail game.
可编译文件和可执行程序放在release里

#徽章
使用Python语言

#运行环境
.exe可以直接在电脑上运行
.py文件需在有Python解释器的开发软件上运行

#编译方法：
PeppaPig.py和Online_game.py文件需放在项目同一目录下
Python好像不用编译（？

#使用方法：直接打开exe文件。
 或者新建项目，将PeppaPig.py和Online_game.py文件放在项目同一目录下
 运行主文件PeppaPig.py，需要Online_game.py文件时会自动调用导入的模块
 然后通过键盘输入输出来交互
